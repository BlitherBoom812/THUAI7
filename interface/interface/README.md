# Interface

## 简介

Unity 客户端

## 目标

### 基本目标

生成 Windows 平台的 Unity 客户端，至少实现以下功能：

- 游戏回放：读取回放文件并回放

### 重要目标

- 直播观战：实时播放当前游戏画面
- 玩家对战：玩家可以自主操控角色，加入游戏

### 提高目标

- 生成适合 Web 的 Unity，在网站上直播赛况（需与网站组配合）
- 生成手机 APP，支持手机客户端直播观战以及实时进行游戏
- 上架 steam
- ......（未完待画）

## 注意事项

编写好适合 Unity 的 .gitignore 文件，将必要的文件上传，既不要上传多余文件，也不要落下必需的文件
